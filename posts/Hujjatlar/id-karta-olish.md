<!-- # O'zbekistonda ID karta qanday olinadi - 2026 -->

ID karta (shaxsni tasdiqlovchi elektron hujjat) - bu yangi avlod biometrik pasport o'rnini bosuvchi zamonaviy hujjat. Bu yo'riqnomada ID karta olish jarayonini qadamma-qadam tushuntiramiz.

## ID karta nima va nima uchun kerak?

ID karta - bu:

- Plastik karta formatidagi shaxsni tasdiqlovchi hujjat
- Ichida elektron chip mavjud
- Pasport o'rnida ishlatiladi
- 10 yil muddatga beriladi

**Afzalliklari:**

- Cho'ntagingizga sig'adi (pasportdan kichikroq)
- Zamonaviy va xavfsiz
- Ko'plab xizmatlar uchun qulay

## Kerakli hujjatlar

1. **Eski pasport** (agar bor bo'lsa)
2. **3x4 o'lchamdagi fotosurat** - 2 dona (oq fonli)
3. **To'lov kvitansiyasi** - 181,000 so'm (2026 yil)
4. **Ro'yxatdan o'tish ma'lumoti** - mahalla yoki IIB dan

## Qadamma-qadam yo'riqnoma

### 1-qadam: my.gov.uz orqali online ariza

1. [my.gov.uz](https://my.gov.uz) saytiga kiring
2. "Xizmatlar" bo'limini tanlang
3. "ID karta olish" ni toping
4. Shaxsiy ma'lumotlaringizni kiriting
5. Ariza yuborish
6. SMS orqali tasdiqlash

**Muhim:** Online ariza berish majburiy emas, lekin jarayonni tezlashtiradi!

### 2-qadam: IIB ga tashrif

1. Yashash joyingizdagi IIB ga boring
2. Navbat oling (online ariza bergan bo'lsangiz tezroq)
3. Hujjatlaringizni topshiring
4. Biometrik ma'lumotlar (barmoq izi, fotosurat) olinadi

**Maslahat:** Ertalab soat 9-10 da kam odam bo'ladi!

### 3-qadam: To'lov

To'lov summasini quyidagi usullar bilan to'lashingiz mumkin:

- Bank kassasi orqali
- Click, Payme orqali
- Terminal orqali

**Narx:** 181,000 so'm (2026 yil)

### 4-qadam: ID kartani olish

- **Tayyor bo'lish muddati:** 10-15 ish kuni
- **Qayerdan olish:** Ariza bergan IIB dan
- SMS xabarnoma keladi

## Tez-tez so'raladigan savollar

**S: ID karta bilan chet elga chiqsa bo'ladimi?**  
J: Yo'q, chet elga chiqish uchun xalqaro pasport kerak. ID karta faqat O'zbekiston ichida ishlatiladi.

**S: Eski pasportim amal qilish muddati tugamagan, ID karta olsam bo'ladimi?**  
J: Ha, ID karta va pasport bir vaqtda bo'lishi mumkin.

**S: ID karta yo'qolsa nima qilish kerak?**  
J: Darhol IIB ga murojaat qiling va yo'qotganligi haqida ariza bering.

## Foydali havolalar

- [my.gov.uz](https://my.gov.uz) - Online ariza
- [id.gov.uz](https://id.gov.uz) - ID karta haqida ma'lumot
- IIB telefon raqamlari - ma'lumotnoma

## Xulosa

ID karta olish jarayoni oddiy va ancha tez. Asosiy qoidalar:

1. Hujjatlarni oldindan tayyorlang
2. Online ariza bering (ixtiyoriy)
3. Erta tongda IIB ga boring
4. 10-15 kun ichida tayyor bo'ladi
